### [2026-07-11] T-cell niche pipeline (Slurm, cpu-medium, ~5 min run)
- **Method (all confirmed with user):** neighbors = ALL cell types (niche=microenvironment); clustering = Leiden. Per T cell i: NE_i(g)=sum_j w_ij*x_j(g), w_ij=exp(-d_ij^2/2sigma^2), normalized; neighbors from KD-tree radius query within sample. 66,825 T cells x 960 genes NE matrix → scale → PCA(30) → kNN(15) → Leiden(res=1.0).
- **Result (success):**
  - **20 T-cell niches (TN0..TN19).**
  - **ARI=0.208, NMI=0.338 vs CellCharter niche** — moderate agreement; new niches SUBDIVIDE the reference.
  - Clean recovery of biology: lymphoid structure (TLS) → TN3 (0.83), TN7 (0.84), TN2 (0.79); immune → TN18 (0.94), TN19 (1.00), TN11 (0.74), TN15 (0.72). CD4 memory dominates immune-mapped niches; CD4 naive dominates TLS-mapped niches.
  - **UNPROMPTED CAVEAT:** spatial maps show a strong per-sample/batch effect — verified from the per-cell assignment table: LUAD-12 is 94.6% TN16, LUSC-6 is 88.8% TN1, LUAD-9 R2 is TN8 56.6%/TN9 42.1%, LUAD-13 is TN11 50.8%/TN15 45.0%. Several samples collapse to one or two niches. Cause: clustering on non-batch-corrected neighborhood expression → sample-level expression differences dominate. Next-step fix candidate: batch-correct NE (harmony on NE PCA) or use neighborhood cell-type COMPOSITION instead of expression, or cluster within-sample then align.
