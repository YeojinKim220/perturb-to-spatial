# Session 2 — T-cell-specific niche 구성 (neighborhood-expression clustering)

User 지시: CellCharter 라벨 재사용 대신 T-cell niche를 처음부터 구성 — (1) T cell별 Gaussian kernel neighborhood expression, (2) clustering(Leiden), 각 cluster = T-cell niche, (3) annotation하고 데이터에 이미 제공된 CellCharter `niche` 라벨과 비교.

### [2026-07-11] 좌표/정규화 diagnostic (Slurm, quick)
- **Result (success):** X는 log-normalized float (max ~12.4, integer 아님); raw counts는 `layers['counts']`. Spatial 좌표: 각 sample이 분리된 좌표 영역을 차지, FOV는 sample 내 tiling → neighbor는 **sample 내** 계산. Median 1-NN 거리 = 46.2 (p90 67.2). → Gaussian **sigma=50**, cutoff **radius=3*sigma=150** 데이터 기반 설정.

### [2026-07-11] T-cell niche pipeline (Slurm, cpu-medium, ~5분 실행)
- **Method (모두 user 확인):** neighbors = 모든 cell type (niche=미세환경); clustering = Leiden. T cell i별: NE_i(g)=sum_j w_ij*x_j(g), w_ij=exp(-d_ij^2/2sigma^2), 정규화; neighbor는 sample 내 KD-tree radius query. 66,825 T cells x 960 genes NE matrix → scale → PCA(30) → kNN(15) → Leiden(res=1.0).
- **Result (success):**
  - **20개 T-cell niche (TN0..TN19).**
  - **CellCharter niche 대비 ARI=0.208, NMI=0.338** — 중간 수준 일치; 새 niche가 reference를 세분화(SUBDIVIDE).
  - 생물학 복원 확인: lymphoid structure(TLS) → TN3 (0.83), TN7 (0.84), TN2 (0.79); immune → TN18 (0.94), TN19 (1.00), TN11 (0.74), TN15 (0.72). immune-mapped niche는 CD4 memory 우세; TLS-mapped niche는 CD4 naive 우세.
