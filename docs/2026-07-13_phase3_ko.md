## 2. Phase-3 core (job 11132307, phase3_core.py) — F1/MI/S2c/REP
- 버그: env_project4에 pyarrow 없음 → parquet를 csv.gz로 교체(job 11131695 실패 후 수정).
- **F1 (독립 state×context):** intrinsic state = T-cell expression only, context = environment only.
  tumor-distance 축만 유효(myeloid/fibro/Bcell-rich tertile은 naive CD4에 거의 없어 degenerate).
  Th1 tumor-contact 농축(logOR+0.28, 3/3), cytotoxic 고갈(−0.86, 3/3), exhausted 비일관.
- **MI (Moran's I 재점검):** 40µm max-dist + cross-FOV edge 제거 + n_counts residualize + BH15.
  **0/15 유의(BH<0.05)** — 기존 "약하지만 유의"는 장거리/cross-FOV edge 아티팩트였음.
- **S2c (stress-corrected contrast):** matched(n_counts) contact-vs-far, section pseudobulk, ≥2/3 방향일치, stress flag.
  top20 중 stress gene 1–4개(기존 HSP/S100/VIM 지배에서 개선). 
  raw-vs-corrected ranking Spearman 0.97–0.99 = ranking이 stress-driven 아니지만, 상위 target은 housekeeping(신호 약함).
- **REP (leave-one-section-out):** 평균 Spearman 0.31(기존 joint-clustering 0.89–0.93은 non-independent라 부풀려짐).
  cytotoxic 최고(0.23–0.56). R1–R3은 한 환자의 section = biological replicate 아님.
